## 6. LINK PLACEMENT & TRAFFIC OPTIMIZATION

### 6.1 Five Methods to Embed Links Without Suppression

**Method #1: First Comment Link (Best)**
- Post text-only
- Immediately comment with link + context
- Reply to your comment to bump it
- Result: Up to 2x reach vs. in-post link

**Method #2: Delayed Edit**
- Post without link
- Wait 2-4 hours until engagement plateaus
- Edit to add link at bottom
- Result: Gets engagement first, link later

**Method #3: Bio/Featured Reference**
- Post text: "Full project in my Featured section"
- Ensure Featured is updated with correct link
- Result: Drives profile visits, then clicks

**Method #4: DM Offer**
- Post text: "DM 'code' and I'll send the link"
- Reply to DMs with link + personal touch
- Result: Drives DMs (algorithm loves DMs), gets link to interested people

**Method #5: Screenshot Method**
- Screenshot the GitHub repo or website
- Include URL visible in screenshot
- Type "URL in the image ðŸ‘†"
- Result: No link suppression, visual proof
